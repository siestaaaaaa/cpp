# 概论

## 命令行启动

Windows：`python`

Mac 或 Linux：`python3`

退出：`exit()`

## 解释器

官方的解释器：CPython，是用 C 语言开发的

IPython 是基于CPython的一个解释器，在交互方式上有所增强

PyPy 是另一个 Python 解释器，采用 JIT 技术对 Python 代码进行动态编译（而非解释），可以显著提高 Python 代码的执行速度。PyPy 和 CPython 是有所不同的

Jython 和 IronPython 分别是运行在 Java 平台和 .Net 平台的解释器，可以把 Python 代码编译成 Java 字节码和 .Net 字节码

## 小技巧

做 Python 开发，可以一边在文本编辑器里写代码，一边在命令行验证

还可以借助 AI 提高开发效率

# 基础知识

## 缩进

Python 使用缩进来组织代码块

建议使用 4 个空格的缩进，可以在文本编辑器设置把 Tab 自动转换为 4 个空格，确保不混用 Tab 和空格

## 字符串

字符串：`''` 或 `""`

多行字符串：`''''''` 或 `""""""`

原始字符串（不转义）：`r''` 或 `r""`

C-Style 格式化字符串：`'%d%f%x%s' % (1, 3.14, 16, 'abc')`

## 布尔运算

布尔值：`True`、`False`

布尔运算：`and`、`or`、`not`

在布尔上下文中，非零数值和非空容器被视为 `True`，零数值和空容器被视为 `False`

## 空值

Python 中用 `None` 表示空

## 除法

浮点数除法：`/`

下取整除法：`//`

## 数据范围

Python 的整数没有大小限制

浮点数也没有大小限制，但是超过一定范围就直接表示为 `inf`

## 变量和对象

Python 的变量相当于指针，数据相当于对象，所以变量相当于指针指向数据对象

## 编码

ASCII：美国的编码标准

Unicode：把所有语言都统一到一套编码

UTF-8：可变长编码，解决 Unicode 的空间浪费问题，兼容 ASCII

目前主流的编码方式：内存中统一使用 Unicode，保存或传输时转换为 UTF-8

`for Elem in Iterable:`

`while Condition:`

# 函数

## 函数名

函数名也是变量，相当于指向函数对象的指针

## 定义函数

定义函数时，需要确定函数名和参数

如有必要，函数内部可以先对参数类型做检查

函数默认返回 `None`

返回多个值，其实是返回一个 tuple

## 默认参数

默认参数一定要指向不可变对象，否则容易出现问题

## 可变参数

`*args` 是可变参数，可以接收任意个参数，自动组装成一个 tuple

可变参数可以直接传入，比如 `func(1, 2, 3)`

也可以先组装成 list 或 tuple，再通过 `*` 传入，比如 `func(*(1, 2, 3))`

## 参数列表顺序

必选参数、默认参数、可变参数

## 汉诺塔问题

```python
# 打印出把所有盘子从柱子A借助柱子B移动到柱子C的过程
def hanoi(n, a, b, c):
    if n == 1:
        print(a, '-->', c)
        return
    hanoi(n - 1, a, c, b)
    hanoi(1, a, b, c)
    hanoi(n - 1, b, a, c)
```

## 闭包

函数内也可以定义函数，内部函数可以引用外部函数的参数和变量（nonlocal）

外部函数可以返回内部函数，其相关的参数和变量都保存在返回的函数中，被返回的函数叫做闭包

每次调用外部函数，返回的闭包都是独立的

## 匿名函数

匿名函数是一个没有名字的函数对象，可以赋给一个变量

`lambda` 表示匿名函数，后面跟参数、`:` 和一个表达式，表达式的结果就是匿名函数的返回值
